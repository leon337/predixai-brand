## Critérios de validação conceitual

- O prompt deve funcionar de forma independente ao ser copiado para uma plataforma de IA.
- A base empresarial deve estar presente no próprio texto, evitando respostas genéricas por falta de contexto.
- Nenhum dado real ou sensível deve ser solicitado.
- Nenhuma ação externa deve ser apresentada como realizada.
- Informações ausentes ou contraditórias devem produzir limite e encaminhamento.
- Conteúdo clínico, interpretação de resultado e decisões financeiras devem permanecer humanos.
- O prompt não deve depender do Supabase ou da API da Vercel nesta etapa.
