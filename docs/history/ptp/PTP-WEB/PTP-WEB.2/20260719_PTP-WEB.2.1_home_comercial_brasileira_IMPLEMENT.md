# PTP-WEB.2.1 — Home Comercial Brasileira

## Objetivo

Reformular a Home da PredixAI BR para que um visitante brasileiro, sem conhecimento prévio do projeto, compreenda nos primeiros segundos:

1. o que é vendido;
2. para quem serve;
3. quais problemas pode resolver;
4. como funciona;
5. qual é a próxima ação.

## Diagnóstico que motivou a correção

A primeira versão técnica do PredixAI Workforce foi aprovada pelos testes automatizados, mas reprovada na validação comercial humana.

```txt
TECHNICAL_VALIDATION=PASS
COMMERCIAL_CLARITY=FAIL
FIRST_10_SECONDS_TEST=FAIL
MERGE_AUTHORIZATION=BLOCKED
```

Os principais problemas encontrados foram:

- Hero institucional e abstrato;
- produto principal pouco evidente;
- categorias `Soluções` e `Produtos` concorrentes;
- excesso de conceitos na Home;
- falta de exemplo concreto;
- uso de termos estrangeiros sem explicação;
- CTA pouco orientado ao problema do cliente.

## Decisão comercial

O PredixAI Workforce passou a ser apresentado como:

```txt
PredixAI Workforce
Equipe inteligente para empresas
```

Mensagem principal:

> Automatize atendimento e tarefas repetitivas sem perder o controle da sua empresa.

O nome da marca permanece, mas toda explicação comercial é direcionada ao público brasileiro e escrita em português do Brasil.

## Nova jornada da Home

```txt
Oferta principal
→ exemplo concreto
→ tarefas que podem ser automatizadas
→ segmentos atendidos
→ implantação progressiva
→ segurança e controle humano
→ diagnóstico inicial
```

## Implementação

### Hero

- Workforce visível imediatamente;
- benefício empresarial direto;
- público brasileiro explícito;
- CTA `Analisar um processo da minha empresa`;
- CTA secundário para conhecer a solução;
- estágio `Em validação` mantido.

### Exemplo concreto

```txt
Cliente envia mensagem
→ assunto é identificado
→ agente consulta informações autorizadas
→ equipe humana assume quando necessário
→ ação fica registrada
```

### Menu

```txt
Workforce
Como funciona
O que automatiza
Para quem
Segurança
Analisar meu processo
```

### Linguagem

A comunicação pública evita categorias vagas e termos estrangeiros desnecessários. `Workforce` permanece somente como nome comercial do produto.
