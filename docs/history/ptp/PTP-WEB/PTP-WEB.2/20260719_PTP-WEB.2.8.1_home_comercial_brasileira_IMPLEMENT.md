# PTP-WEB.2.8.1 — Home Comercial Brasileira

## Objetivo

Reformular a Home da PredixAI BR para que um visitante brasileiro, sem conhecimento prévio, compreenda rapidamente:

1. o que é vendido;
2. para quem serve;
3. quais problemas pode resolver;
4. como funciona;
5. qual é a próxima ação.
