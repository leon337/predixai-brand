# PTP-WEB.2.8.1 — Home Comercial Brasileira

## Objetivo

Reformular a Home da PredixAI BR para que um visitante brasileiro, sem conhecimento prévio, compreenda rapidamente:

1. o que é vendido;
2. para quem serve;
3. quais problemas pode resolver;
4. como funciona;
5. qual é a próxima ação.

## Nota de governança

A correção foi aprovada na conversa como `PTP-WEB.2.1 ajustada`. Como `PTP-WEB.2.1` já identificava a arquitetura comercial no roadmap, o código canônico foi normalizado para:

```txt
PTP-WEB.2.8.1
Nome curto: Home Comercial Brasileira
```

## Diagnóstico inicial

```txt
TECHNICAL_VALIDATION=PASS
```

Problemas confirmados:

- categorias concorrentes;
- excesso de conceitos;
- falta de exemplo concreto;
- termos estrangeiros sem explicação;
- CTA pouco orientado ao problema empresarial;

## Decisão comercial

```txt
PredixAI Workforce
Equipe inteligente para empresas
```

Mensagem principal:

> Automatize atendimento e tarefas repetitivas sem perder o controle da sua empresa.

## Nova jornada

```txt
Oferta principal
→ exemplo concreto
→ tarefas automatizáveis
→ empresas brasileiras e segmentos
→ implantação progressiva
→ segurança e controle humano
→ diagnóstico inicial
```

## Alterações comerciais

- Workforce centralizado no Hero;
- link do Workforce no menu, Hero e rodapé;
- público brasileiro declarado explicitamente;
- CTA principal alterado para `Analisar um processo da minha empresa`;
- menu simplificado;
- categorias `Soluções` e `Produtos` removidas da navegação;
- exemplo concreto de funcionamento;
- processos automatizáveis descritos em linguagem empresarial;
- segmentos e estágios reais reorganizados;
- termos estrangeiros públicos removidos, exceto marcas consolidadas;
- segurança e supervisão humana preservadas.

## Compatibilidade de publicação
