## Regra de sincronização obrigatória

Ao concluir cada etapa:

1. Marcar o item correspondente na issue `LEA-116`.
2. Atualizar este documento no GitHub.
3. Registrar arquivos técnicos e documentais afetados.
4. Registrar validações executadas.
5. Registrar commit SHA e PR.
6. Registrar impacto no Supabase e deployment da Vercel, quando houver.
7. Não declarar conclusão sem evidência publicada.
