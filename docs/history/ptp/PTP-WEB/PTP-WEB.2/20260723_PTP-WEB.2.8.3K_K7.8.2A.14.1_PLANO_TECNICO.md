## 10. Origem de `promptInstruction`

Nesta mini-PTP, as instruções semânticas serão resolvidas por um mapa versionado no materializador da clínica.

```txt
PROMPT_INSTRUCTION_SOURCE=VERSIONED_CLINIC_MATERIALIZER_MAP
PROMPT_INSTRUCTION_MAP_VERSION=1
MISSING_PROMPT_INSTRUCTION=BLOCK
UNUSED_PROMPT_INSTRUCTION=FAIL_OR_REVIEW
```

A suíte deverá comprovar cobertura de todas as opções usadas pelo pacote.

## 19. Riscos e controles

| Risco | Nível | Controle |
|---|---:|---|
| perda de regra canônica | alto | parser contratual + matriz objetiva |
| divergência entre UI e prompt | alto | mesma `EffectiveAgentConfig` |
| prototype pollution | alto | allowlist, propriedades próprias e segmentos proibidos |
| hash inconsistente | alto | SHA-256 + serialização canônica versionada |
| `promptInstruction` ausente | alto | mapa versionado com cobertura total |
| regressão do criador genérico | alto | projeção opcional + teste dedicado |
| alteração silenciosa de texto | médio | detectar e bloquear |
| quebra de sessão | médio | migração mínima + `lastKnownGood` |
| generalização prematura | médio | materializador específico da clínica |
