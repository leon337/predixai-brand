## 2. Objetivo

Criar um site institucional futurista, organizado, responsivo e alinhado à identidade ampla da PredixAI BR, publicando o código em repositório próprio e preparando hospedagem contínua no GitHub Pages.

## 3. Decisão arquitetural

Foi adotada arquitetura estática sem dependências runtime:

```txt
HTML5 semântico
→ CSS moderno e responsivo
→ JavaScript progressivo
→ ativos SVG
→ GitHub Actions
→ GitHub Pages
```

A opção reduz superfície de ataque, custo de manutenção, tempo de carregamento e dependência de build para um site institucional.

## 4. Conteúdo e UX

O site inclui:

- apresentação institucional;
- soluções em agentes, automação, visão/OCR, decisão, produtos e educação;
- método PredixAI em cinco etapas;
- ecossistema Platform, Academy e Lab;
- princípios de clareza, segurança e evidência;
- chamada para novos projetos;
- navegação mobile-first;
- design futurista com glassmorphism controlado;
- animações com fallback para movimento reduzido.
