# PTP-WEB.1.6.2V1 — Isolamento do Deploy Vercel

## Objetivo

Preparar o repositório `leon337/predixai-brand` para publicação na Vercel sem expor a memória técnica do projeto.

## Decisão arquitetural

A Vercel deve executar:

```txt
bash scripts/build_vercel_static.sh
```

O script cria `dist/` e copia somente os arquivos públicos do site.

## Arquivos criados

- `scripts/build_vercel_static.sh`
- `vercel.json`
- `reports/20260719_PTP-WEB.1.6.2V1_isolamento_preview_vercel_IMPLEMENT.txt`

## Conteúdo público permitido

```txt
index.html
404.html
manifest.webmanifest
robots.txt
sitemap.xml
.nojekyll
assets/
solucoes/       # quando existir
validacao/      # quando existir
privacidade/    # quando existir
obrigado/       # quando existir
```

## Conteúdo bloqueado no deployment

```txt
docs/
reports/
.github/
PROJECT_STATE.md
predixai_context.json
README.md
.git/
.env*
*.pem
*.key
arquivos com credential, secret ou token no nome
```

## Configuração Vercel

```txt
Application Preset: Other
Root Directory: ./
Build Command: bash scripts/build_vercel_static.sh
Output Directory: dist
Install Command: automático ou vazio
Environment Variables: nenhuma
```
